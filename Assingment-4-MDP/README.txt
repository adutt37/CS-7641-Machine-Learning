This project seeks to understand the three reinforcement learning algorithms by applying them each to two different Markov decision processes (MDP). 
The reinforcement learning methods are value iteration, policy iteration, and Q-learning.The two MDP problems forzen lake and forest management.


Getting Started
Prerequisites
Ensure you have Python installed on your machine.

Required libraries may include pandas, numpy, scikit-learn, and others. You can install any missing dependencies 
using pip install -r requirements.txt if a requirements file is provided.
Installation
Clone this repository to your local machine:

git clone https://github.com/adutt37/CS-7641-Machine-Learning.git
Navigate to the project directory:

cd CS-7641-Machine-Learning/Assingment-4-MDP

Running the Scripts
Run the main Python script:

py forest_management.py
py forzen_lake.py
